# D-mart-project

A simple Python console-based Dmart billing system that takes customer details, product quantity, price, GST, and discount to generate a final invoice.

## Sample output

```
==================================================
============ Dmart =============
==================================================
============ Bill =============
bill no : 101
date    : 2026-07-14 12:34:56
Customer Name: Rahul Kumar
==================================================
Product_name	Qty	 unit_price	   amount
Apple	3	       3    30.00	       90.00
Milk	2	       2    50.00	       100.00
Bread	1	       1    40.00		     40.00
==================================================
============ invoice =============
bill amount  : 230.00
gst amount   : 41.40
total amount : 271.40
```

_Note: Above output is a sample run (date/time shown is illustrative)._
